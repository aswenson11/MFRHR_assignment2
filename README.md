
This repository includes data and analysis folders, named accordingly. 

In this analysis I first investigated the data and found that the 'smoke' variable is likely indicating whether the individual is a current smoker.

I then found that this data is likely 5000 individuals rather than a set of individuals tracked over time.

Finally, I used a linear regression to find that, while smoking and age both predict higher costs, the interaction between the two seems insignificant.
